# Code with C

A comprehensive collection of C programming examples and exercises, covering fundamental to intermediate concepts.

## 📚 Contents

This repository contains organized C programming code examples and assessments:

- **01_NOTES** - Educational notes and documentation
- **assessments** - Programming assessment exercises
- **control_flow** - Loop concepts and conditional statements
- **fundamentals** - Basic C programming concepts
- **utilities** - Helper functions and common utilities

## 🚀 Getting Started

### Prerequisites
- GCC compiler or any C compiler
- Basic knowledge of C programming

### Building

To compile any C program:
```bash
gcc -o output_name source_file.c
./output_name
```

Or compile with debugging info:
```bash
gcc -g -o output_name source_file.c
```
